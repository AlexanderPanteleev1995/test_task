Тестовое задание
#### При запуске playbook.yml, ansible копирует папку srv в корень виртуальной машины, где лежат все конфигурационные файлы, устанавливает docker и запускает docker-compose. 
#### Структура конфигурационных файлов:
* /srv/nginx/conf
* /srv/apache/conf
* /srv/mysql/conf
#### Структура логов:
* /srv/nginx/logs
* /srv/apache/logs
* /srv/mysql/logs
#### Доменные имена:
* first.ru
* second.ru
* grafana.ru
* prometheus.ru
